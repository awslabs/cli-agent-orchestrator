Health Check

Verify the server is running and reachable.

GET /health Server liveness probe ▾

List Sessions

Retrieve all active sessions managed by the orchestrator.

GET /sessions List all sessions ▾

Create Session

Start a new tmux session with one agent terminal.

POST /sessions Create session + terminal ▾

Get Session

Retrieve details of a specific session by name.

GET /sessions/{session_name} Session details ▾

Delete Session

Delete a session and all its terminals.

DELETE /sessions/{session_name} Terminate session ▾

List Terminals in Session

Retrieve all terminals belonging to a session.

GET /sessions/{session_name}/terminals List session terminals ▾

Add Terminal to Session

Create an additional terminal inside an existing session.

POST /sessions/{session_name}/terminals Add terminal to session ▾

Get Terminal

Retrieve details of a terminal by its 8-character hex ID.

GET /terminals/{terminal_id} Terminal details ▾

Get Working Directory

Retrieve the current working directory of a terminal's pane.

GET /terminals/{terminal_id}/working-directory Terminal working dir ▾

Send Input to Terminal

Send a message/command to a running terminal.

POST /terminals/{terminal_id}/input Send input text ▾

Get Terminal Output

Read the terminal's captured output log.

GET /terminals/{terminal_id}/output Terminal output log ▾

Exit Terminal

Send a provider-specific exit command to gracefully stop the agent.

POST /terminals/{terminal_id}/exit Send exit command ▾

Delete Terminal

Permanently remove a terminal from the session.

DELETE /terminals/{terminal_id} Remove terminal ▾

Send Inbox Message

Queue a message from one terminal to another. Delivered when the receiver is idle.

POST /terminals/{receiver_id}/inbox/messages Send inbox message ▾

Get Inbox Messages

Retrieve queued or delivered inbox messages for a terminal.

GET /terminals/{terminal_id}/inbox/messages List inbox messages ▾
